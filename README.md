Digital Clock
A simple digital clock built using HTML, CSS, and JavaScript. The clock displays the current time and updates automatically every second.

Features
    .Real-time clock updates
    .Clean and responsive design
    .Built with vanilla javaScript
    .Beginner-friendly project
    
Technologies Used
   .HTML5
   .CSS3
   .JavaScript
   
How It Works
JavaScript uses the Date object to get the current time and updates the clock display every second using setInterval().
